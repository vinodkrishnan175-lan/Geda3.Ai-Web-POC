# Geda3.Ai Workshop TAT PoC

Files:
- streamlit_app.py
- tat_engine.py
- requirements.txt

## Local run
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Streamlit Community Cloud
1. Push these files to your GitHub repository.
2. In Streamlit Community Cloud, connect your GitHub account and select the repository / branch.
3. Set the main file path to `streamlit_app.py`.
4. Keep `requirements.txt` in the repo root so Community Cloud installs the dependencies automatically.
